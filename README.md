# Simple-Expense-Calculator-
This Python Expense Tracker allows users to add, view, and categorize daily expenses, storing data in a file. It calculates totals per category, helping manage personal finances efficiently and simply.
# Features
* Add additional costs along with the category, amount, and optional note.
* View every expense that has been recorded.
* Determine the total amount spent in each category.
* keeps information persistent by storing it in a text file.
# Installation
1. Ensure Python 3 is installed on your system.
2. Clone the repositary
3. Run the code
# Usage
* Run the program
* Choose an option from the menu:
Add Expense,
View All Expenses,
View Total by Category,
Exit
* Follow on-screen prompts to enter expense details
